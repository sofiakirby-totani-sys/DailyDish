## General info
DailyDish is a GUI application with Python that has imported TKinter, which allows to randomy suggest dinner meal ideas which the user can choose to have or not. If user accepts then it displays the full ingredient checklist with an additional embedded URL to access the full recipe. The purpose is to reduce decision fatigue, streamline meal planning and avoid forgetting items during the shopping process. 

## Technologies
Project is created with:

**Languages**
    Python 3

**Libraries**
    tkinter- Python interface for the GUI toolkit 
    random- Python module that allows for random number/selection operations 
    webbrowser-Python module to open URLs in a browser

**Versions**
  random and webrowser are part of Python standard library which means it responds to the Python version 3
  this has been tested on macOS Python 3.0
  

## Features
- Randomly generated meal ideas
- Shopping list with checkboxes
- Embedded URL for full recipe link
